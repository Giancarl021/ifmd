# preview

Create a Web Server

## Usage

```bash
ifmd preview <path/to/file> <flags>
```

## Flags

-   `-p` | `--port`: The port for the web server to listen to. Defaults to `3000`. Value: `<number>`;
-   `-d` | `--date`: Sets the date of the documents, defaults to `Date.now()`. Value: `<date verbatim>`.
